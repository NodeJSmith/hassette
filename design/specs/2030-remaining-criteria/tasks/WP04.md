## Objectives & Success Criteria

- All 8 telemetry endpoints in `telemetry.py` accept optional `session_id: int | None = Query(default=None)` instead of calling `safe_session_id()`.
- The `/bus/listeners` endpoint in `bus.py` accepts the same parameter.
- `RuntimeDep` is removed from the 6 endpoints that no longer use it (`app_health`, `app_listeners`, `app_jobs`, `handler_invocations`, `job_executions`, `dashboard_errors`).
- `safe_session_id()` in `telemetry_helpers.py` has a comment noting `ws.py` as the remaining consumer.
- Module docstring in `telemetry.py` is updated.
- All existing integration tests pass with the new signatures.

## Subtasks

1. In `src/hassette/web/routes/telemetry.py`, add `session_id: int | None = Query(default=None)` parameter to all 8 telemetry endpoints. Remove the `session_id = safe_session_id(runtime)` line from each. Pass the query param directly to the query service calls.
2. Remove `runtime: RuntimeDep` from the 6 endpoints that no longer use it: `app_health`, `app_listeners`, `app_jobs`, `handler_invocations`, `job_executions`, `dashboard_errors`. Keep it on `dashboard_kpis` and `dashboard_app_grid`.
3. Remove the import of `safe_session_id` from `telemetry.py` (and `RuntimeDep` if fully unused ‚Äî check `dashboard_kpis` and `dashboard_app_grid` still need it).
4. Update the module docstring at the top of `telemetry.py` to reflect client-provided session scoping.
5. In `src/hassette/web/routes/bus.py`, add `session_id: int | None = Query(default=None)` to `get_listener_metrics()`. Remove `safe_session_id()` call. Remove `runtime: RuntimeDep` if no longer needed. Pass `session_id` to both `gather_all_listeners()` and `get_listener_summary()`.
6. In `src/hassette/web/telemetry_helpers.py`, add a comment to `safe_session_id()`: "Remaining consumer: ws.py connected_payload_from(). Telemetry endpoints no longer use this ‚Äî session_id is client-provided."
7. Update `tests/integration/test_web_api.py` ‚Äî existing tests that mock or inject `RuntimeDep` for the 6 modified endpoints need updating. Add tests verifying session_id query param works (pass explicit session_id, verify scoped results; omit it, verify all-time results).
8. Update any bus endpoint tests in `tests/integration/` if they exist.

## Test Strategy

- **Integration tests** in `tests/integration/test_web_api.py`:
  - Update existing tests to not provide `RuntimeDep` for the 6 endpoints where it was removed
  - `test_telemetry_endpoint_session_id_param` ‚Äî for each endpoint, verify `?session_id=<N>` returns session-scoped data
  - `test_telemetry_endpoint_no_session_id_returns_all_time` ‚Äî verify omitting `session_id` returns all-time data
  - `test_bus_listeners_session_id_param` ‚Äî verify `/bus/listeners?session_id=<N>` works
