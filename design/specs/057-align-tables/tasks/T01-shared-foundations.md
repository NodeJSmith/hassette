## Summary
Create the foundational shared components and types that all subsequent tasks depend on: extract `ColumnFilterPopover` to shared (with focus management fix), create `FilterIcon`, create `ColumnFilters` type, create `TableFooter`, and update `ht-table` CSS defaults. This task builds the shared infrastructure — no page migration yet.

## Prompt
1. **Create `components/shared/table-types.ts`** — define `ColumnFilter` and `ColumnFilters` types per Architecture §3 in the design doc.

2. **Extract `ColumnFilterPopover`** — move `components/shared/log-table/column-filter.tsx` and `column-filter.module.css` to `components/shared/column-filter-popover/index.tsx` + `index.module.css`. Update imports in `log-table-header.tsx`, `log-table-footer.tsx`, and `log-table/column-picker.tsx` (which also imports it). Fix focus management during extraction per Architecture §1: on open, focus first focusable child; on close, restore focus to `triggerRef.current`; trap Tab/Shift-Tab within popover.

3. **Create `components/shared/filter-icon.tsx`** — extract the inline funnel SVG from `log-table-header.tsx` line 79–81 into a shared component per Architecture §2.

4. **Create `components/shared/table-footer.tsx` + `.module.css`** — shared footer per Architecture §6. Props: `count`, `columnFilters?`, `onResetFilters?`, `extras?`. On mobile, auto-generate the filter panel from `columnFilters`. Use the styling pattern from `log-table-footer.module.css` (`.footer`, `.count`).

5. **Update `styles/tables.css`** — per Architecture §7: add `table-layout: fixed` to `.ht-table`, change `td` `vertical-align` to `top`, add `overflow: hidden` and `text-overflow: ellipsis` on `td`. Remove `table-layout: fixed` from `.ht-table--compact` (now redundant). Remove `ht-table-toolbar` and all related styles (`.ht-table-toolbar__title`, `.ht-table-toolbar__heading`, `.ht-table-toolbar__note`, `.ht-table-toolbar__controls`).

6. **Write tests** — co-located `.test.tsx` files for `ColumnFilterPopover` (focus-on-open, focus-restore, Tab trap, Escape close, click-outside close), `FilterIcon` (renders SVG, accepts size prop), and `TableFooter` (count display, mobile filter panel open/close from columnFilters, extras slot rendering). Use Vitest + @testing-library/preact.
