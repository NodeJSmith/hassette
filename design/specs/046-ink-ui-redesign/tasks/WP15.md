### 1. Logs table: age column

Add an "age" column to the log table (`components/shared/log-table.tsx`) showing relative time (e.g., "12s", "3m", "1h"). Use the existing `useRelativeTime` hook or compute inline from the log entry timestamp. Position it as the last column.

On mobile where columns are condensed, the age column can replace the absolute timestamp (show relative time instead of absolute).
