## Objectives & Success Criteria

Stream 5: Extract the 859-line `tests/e2e/conftest.py` into composable modules, deduplicate fixtures shared with integration conftest, and remove legacy snapshot factories if unused.

Success:
- `mock_hassette` fixture and its seed-data builders are extracted to `tests/e2e/mock_fixtures.py`
- `tests/e2e/conftest.py` is under 400 lines
- `runtime_query_service` and `app` fixtures shared between `tests/integration/conftest.py` and `tests/e2e/conftest.py` are deduplicated — defined in one place, imported by both
- Legacy snapshot factories (`make_old_app_instance`, `make_old_snapshot`) are removed if unused, or migrated if still referenced
- E2E test suite passes; no regressions in integration tests

## Subtasks

1. Create `tests/e2e/mock_fixtures.py`: extract `_build_manifests()`, `mock_hassette` fixture, and all seed-data construction (telemetry listeners, jobs, sessions, app instances, error records) from `conftest.py` into composable factory functions. The `mock_hassette` fixture itself stays in `conftest.py` (pytest discovers fixtures in conftest) but calls the factory functions
2. Extract telemetry mock wiring functions from `mock_hassette` into named helpers in `mock_fixtures.py`: `wire_listener_telemetry(hassette, ...)`, `wire_job_telemetry(hassette, ...)`, `wire_session_telemetry(hassette, ...)`, `wire_error_telemetry(hassette, ...)`, `wire_global_summary(hassette, ...)`
3. Deduplicate `runtime_query_service` and `app` fixtures: these are defined in both `tests/integration/conftest.py` (lines 106-114) and `tests/e2e/conftest.py` (lines 735-802). Move the shared definitions to a common location (e.g., `src/hassette/test_utils/web_helpers.py` or a shared conftest) and import from both
4. Check if `make_old_app_instance` and `make_old_snapshot` (re-exported from `test_utils.__init__.py`) are used anywhere in the codebase. If unused, remove them from `web_helpers.py` and the `__init__.py` re-exports. If used, keep as-is
5. Verify `tests/e2e/conftest.py` is under 400 lines after extraction
