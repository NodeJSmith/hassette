## Objectives & Success Criteria

Visual foundation for all pages. After this WP:
1. `tokens.css` implements the full Graphite + Emerald palette from `design/direction.md` with `[data-theme="light"]` and `[data-theme="dark"]` selectors
2. `base.html` renders the root layout: Google Fonts (Space Grotesk, JetBrains Mono, DM Sans), HTMX, Alpine.js, idiomorph, Lucide icon pattern, sidebar, theme toggle
3. `nav.html` renders the 56px icon rail with same-temperature background, nav items, and breathing pulse dot
4. Theme toggle persists preference in `localStorage`
5. The page renders correctly in both light and dark modes with no raw hex values in `style.css`

## Subtasks

1. Rewrite `src/hassette/web/static/css/tokens.css` with all `--ht-*` custom properties from `design/direction.md`. Light mode in `[data-theme="light"]`, dark mode in `[data-theme="dark"]`. Include typography scale, spacing scale, radius scale, shadow values, motion values.

2. Create initial `src/hassette/web/static/css/style.css` with:
   - Reset and base body styles
   - `.ht-layout` grid (56px sidebar + 1fr main)
   - `.ht-sidebar` styles (same-temperature, border-right, sticky, flex column)
   - `.ht-nav-item` styles (hover, active with accent)
   - `.ht-pulse-dot` with `breathe` animation and `.disconnected` variant
   - `.ht-main` content area padding
   - Theme toggle positioning and styling
   - Responsive breakpoint (sidebar hidden below 768px)
   - `prefers-reduced-motion` media query

3. Rewrite `src/hassette/web/templates/base.html`:
   - Google Fonts link (Space Grotesk, JetBrains Mono, DM Sans)
   - Drop Font Awesome CDN
   - HTMX 2.0.4, idiomorph 0.3.0, Alpine.js 3.14.8 (same versions)
   - Link `tokens.css` and `style.css`
   - `data-theme` attribute on `<html>` (default dark, read from `localStorage`)
   - Layout grid with sidebar + main
   - Theme toggle button with sun/moon SVG icons
   - `{% block content %}` inside main area

4. Rewrite `src/hassette/web/templates/components/nav.html`:
   - 4 nav items: Dashboard, Apps, Logs (3 SVG icons from Lucide)
   - Brand icon in sidebar top
   - Pulse dot at sidebar bottom with `data-testid="pulse-dot"`
   - Active state from `active_page` context variable
   - `data-testid` on each nav item

5. Add theme toggle JS inline in `base.html`: read `localStorage`, set `data-theme`, toggle on click, persist.

## Test Strategy

E2E test file: `tests/e2e/test_navigation.py`

- `test_sidebar_active_state` — active nav item has accent styling on each page
- `test_pulse_dot_visible` — pulse dot element is visible in sidebar
- `test_theme_toggle_switches_mode` — clicking toggle changes `data-theme` attribute
- `test_theme_persists_across_reload` — set dark, reload, still dark

## Review Guidance

- Every CSS value in `style.css` must reference a `--ht-*` token. Grep for raw hex values — any found is a blocker.
- Token names and values must match `design/direction.md` exactly.
- No Font Awesome references anywhere in `base.html`.
- Lucide icons are inline SVGs, not loaded from CDN.
- Sidebar must use `--ht-bg` (same temperature as page), not a dark background.
