## Objectives & Success Criteria

After this WP:
1. `TelemetryQueryService.get_all_app_summaries()` returns `dict[str, AppHealthSummary]` via 2 SQL queries (not 2N)
2. `get_global_summary()` returns `GlobalSummary | None` (not `dict | None`)
3. `get_current_session_summary()` returns `SessionSummary | None` (not `dict | None`)
4. `compute_app_grid_health()` in context.py calls the batch method instead of looping
5. Dashboard route and dashboard-app-grid partial use the batch method
6. Dashboard template uses attribute access (not `.get()` dict access)
7. All existing tests pass

## Subtasks

1. Add `AppHealthSummary` Pydantic model to `src/hassette/core/telemetry_models.py` with fields: `handler_count: int`, `job_count: int`, `total_invocations: int`, `total_errors: int`, `total_executions: int`, `total_job_errors: int`, `avg_duration_ms: float`, `last_activity_ts: float | None`

2. Add `get_all_app_summaries(session_id: int | None = None) -> dict[str, AppHealthSummary]` to `src/hassette/core/telemetry_query_service.py`. Two SQL queries with `GROUP BY app_key` and `WHERE instance_index = 0`. First query: listener stats per app. Second: job stats per app. Merge results into `AppHealthSummary` per app_key.

3. Audit `SessionSummary` fields against DB schema. If `last_heartbeat_at` can be NULL in the `sessions` table, change to `float | None`.

4. Update `get_global_summary()` to return `GlobalSummary | None`. Handle fresh-install edge case: when SQL returns no rows, return `GlobalSummary(listeners=ListenerGlobalStats(total_listeners=0, invoked_listeners=0, total_invocations=0, total_errors=0, total_di_failures=0, avg_duration_ms=None), jobs=JobGlobalStats(total_jobs=0, executed_jobs=0, total_executions=0, total_errors=0))`.

5. Update `get_current_session_summary()` to return `SessionSummary | None`.

6. Simplify `compute_app_grid_health()` in `src/hassette/web/ui/context.py` to call `get_all_app_summaries()` once instead of looping per-app.

7. Update `src/hassette/web/ui/router.py` dashboard route and `src/hassette/web/ui/partials.py` dashboard-app-grid partial to use the batch method.

8. Update `src/hassette/web/templates/pages/dashboard.html` to use attribute access on `global_summary` and `session_summary` instead of `.get()` dict access.
