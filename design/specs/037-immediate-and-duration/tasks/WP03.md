## Objectives & Success Criteria

Handlers registered with `immediate=True` fire at registration time when the current entity state matches predicates. Synthetic event has `old_state=None`, `time_fired=date_utils.now()`, unique `HassContext`. `P.AttrDidChange` treats `old_state=None` as a match. All immediate-fire tests pass.

## Subtasks

1. Modify `P.AttrDidChange` in `src/hassette/event_handling/predicates.py` to return True (match) when `old_state is None` ‚Äî attribute present equals "considered changed" for bootstrap purposes
2. Implement `_spawn_immediate_fire_task()` in `src/hassette/core/bus_service.py`:
   - Called after `_register_then_add_route()` completes route insertion + DB registration
   - Spawned as separate `task_bucket.spawn()` task (not inline in registration)
   - Reads current state from `StateProxy` via direct dict access: `self.hassette._state_proxy.states.get(entity_id)` ‚Äî no `get_state()`, no retry, no exception
   - If state is None ‚Üí return silently, log at DEBUG
   - Constructs synthetic `RawStateChangeEvent` with `old_state=None`, `new_state=current`, `time_fired=date_utils.now()`, `context=HassContext(id=str(uuid4()), parent_id=None, user_id=None)`
   - Runs predicates via `listener.matches(synthetic_event)` ‚Äî if match, dispatches via `_make_tracked_invoke_fn()`
   - Entire task wrapped in `try/except Exception` ‚Üí log WARNING on failure
   - `ResourceNotReadyError` specifically ‚Üí log ERROR (sequencing invariant violation)
3. Wire `_spawn_immediate_fire_task()` call at the end of `_register_then_add_route()` when `listener.immediate is True`
4. Handle `once=True` + `immediate=True`: if immediate fires, `listener.mark_fired()` is called via `listener.dispatch(invoke_fn)` which already handles the once guard

## Test Strategy

Integration tests in `tests/integration/test_bus_immediate.py` (new file):
- `test_immediate_fires_when_state_matches` ‚Äî entity in target state at registration ‚Üí handler fires with synthetic event
- `test_immediate_no_fire_when_state_does_not_match` ‚Äî entity in non-target state ‚Üí no fire
- `test_immediate_no_fire_entity_not_found` ‚Äî entity not in StateProxy ‚Üí no fire, no error
- `test_immediate_synthetic_event_structure` ‚Äî old_state=None, new_state=current, time_fired is ZonedDateTime, context.id is unique UUID
- `test_immediate_with_once_consumes_invocation` ‚Äî immediate fires, subsequent live event does NOT fire
- `test_immediate_with_debounce` ‚Äî immediate fire passes through debounce guard
- `test_immediate_glob_entity_rejected` ‚Äî ValueError on registration
- `test_immediate_attribute_change_with_attr_did_change` ‚Äî AttrDidChange returns True when old_state=None
- `test_immediate_changed_false_fires_for_any_existing_entity` ‚Äî no state-change predicate, fires if entity exists

Unit test for AttrDidChange in `tests/unit/bus/test_predicates.py`:
- `test_attr_did_change_returns_true_when_old_state_none` ‚Äî old_state=None ‚Üí True (bootstrap match)
