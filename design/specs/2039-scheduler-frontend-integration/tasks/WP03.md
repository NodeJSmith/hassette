## Objectives & Success Criteria

- `job-row.tsx` renders `trigger_label` as primary subtitle and `trigger_detail` as secondary (dimmed), replacing the fallback chain.
- Group pill (`ht-badge`) is visible when `job.group` is non-null; clicking it invokes `onGroupClick` callback.
- Jitter tag ("¬±{N}s") is visible when `job.jitter` is non-null.
- Cancelled jobs render with a muted/struck-through row treatment (durable ‚Äî backed by DB `cancelled_at`).
- When `next_run` is present, expanded detail shows "Next: fires in {relative time}" (and "¬±{jitter}s jitter" if applicable). Uses `useRelativeTime` with the epoch float directly.
- New `job-row.test.tsx` covers all rendering states.

## Subtasks

1. Add `onGroupClick?: (group: string) => void` prop to `JobRow` component interface.
2. Replace the subtitle fallback chain (`trigger_detail ?? trigger_label ?? trigger_type`) with structured rendering: `trigger_label` as primary text; when `trigger_label` is empty string, fall back to `trigger_type`. Show `trigger_detail` as secondary dimmed span (`ht-text-muted`) when non-null.
3. Add conditional subtitle line 2: render `group` as a clickable `ht-badge` pill (calls `onGroupClick(job.group)` on click). Render `jitter` as "¬±{jitter}s" tag when non-null. Render `cancelled` as a badge or apply `ht-text-muted` + strikethrough CSS class to the entire row.
4. In the expanded detail section (below execution history), add a "Next: fires in {relative time}" line when `next_run` is non-null. When `jitter` is also non-null, append "(¬±{jitter}s jitter)". Use `useRelativeTime(job.next_run ?? null)` ‚Äî the `?? null` is needed because TypeScript optional fields arrive as `undefined`, but `useRelativeTime` expects `number | null`. `next_run` is an epoch float matching `last_executed_at`.
5. Write `frontend/src/components/app-detail/job-row.test.tsx` following `components/dashboard/app-card.test.tsx` pattern.

## Review Guidance

- The subtitle MUST NOT use a fallback chain. `trigger_label` is the primary; `trigger_detail` is always secondary/dimmed.
- `onGroupClick` callback is optional ‚Äî when absent, the pill should still render (just not clickable). This lets `job-row.tsx` work in any context.
- `cancelled` treatment must be durable (not just a live transient hint). DB `cancelled_at` backs this.
- `useRelativeTime` receives an epoch float (number), not a string.
