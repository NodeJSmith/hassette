The `ScheduledJob` default for `name` is `name: str = field(default="", compare=False)` — do NOT change this default in this task. T02 handles making `name` required at the Scheduler API level. This task only removes `name_auto` and auto-derivation.

The migration file numbering: 009.sql is the latest. Name the new file `010.sql`.
