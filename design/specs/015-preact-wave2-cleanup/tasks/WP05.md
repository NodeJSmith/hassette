## Objectives & Success Criteria

- Relative timestamps ("2m ago", "1h ago") update every 30 seconds without user interaction
- New `useRelativeTime(timestamp)` hook encapsulates the tick subscription
- `tick: signal(0)` added to `AppState`, incremented via `setInterval` with proper cleanup
- All 4 call sites of `formatRelativeTime` migrated to `useRelativeTime`

## Subtasks

1. Add `tick: signal(0)` to `createAppState()` in `frontend/src/state/create-app-state.ts`.
2. In `frontend/src/app.tsx`, add a `useEffect` in the `App` component that starts a 30-second `setInterval` incrementing `state.tick.value`. Return `clearInterval` in the cleanup function.
3. Create `frontend/src/hooks/use-relative-time.ts`:
   ```typescript
   export function useRelativeTime(timestamp: number | null): string {
     const { tick } = useAppState();
     void tick.value; // subscribe to periodic updates
     return formatRelativeTime(timestamp);
   }
   ```
4. Update `frontend/src/components/app-detail/handler-row.tsx`: replace `formatRelativeTime(listener.last_invoked_at)` (line 77) with `useRelativeTime(listener.last_invoked_at)`. Update imports.
5. Update `frontend/src/components/app-detail/job-row.tsx`: replace `formatRelativeTime(job.last_executed_at)` (line 78) with `useRelativeTime(job.last_executed_at)`. Update imports.
6. Update `frontend/src/components/dashboard/app-card.tsx`: replace `formatRelativeTime(app.last_activity_ts)` (line 44) with `useRelativeTime(app.last_activity_ts)`. Update imports.
7. Update `frontend/src/components/dashboard/error-feed.tsx`: replace `formatRelativeTime(err.timestamp)` (line 23) with `useRelativeTime(err.timestamp)`. Update imports.
8. Update `frontend/src/state/create-app-state.test.ts`: add `tick` independence assertion.

## Test Strategy

- **`frontend/src/state/create-app-state.test.ts`**: Assert `a.tick.value` starts at 0, `a.tick.value = 5` does not affect `b.tick.value`.
- **Manual verification**: Open the app, wait 30+ seconds, observe that relative timestamps change without interaction.

## Review Guidance

- Verify `useRelativeTime` reads `tick.value` to create the signal dependency (without this, timestamps won't update)
- Verify `clearInterval` is called in the `useEffect` cleanup (prevents interval leaks on HMR)
- Verify all 4 `formatRelativeTime` call sites are migrated (grep for direct imports should return 0)
- BLOCKER if: `formatRelativeTime` is still called directly in any component (bypass tick subscription)
