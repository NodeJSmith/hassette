- All existing tests pass

## Review Guidance

- **BLOCKER**: `mark_registered()` must use `LOGGER.warning` + early return, NOT `RuntimeError` — the design explicitly chose resilience over strictness for app reload safety
- **BLOCKER**: No bare `listener.db_id = ...` or `job.db_id = ...` in production code (`src/`) — grep to verify
- **BLOCKER**: `_validate_options()` must be the sole validation path — no duplicate `if debounce is not None and debounce <= 0` in `Listener.create()` body
- **WARNING**: Test files that set `db_id = None` on mocks (test_registration.py) need special handling — they may need to construct the mock without calling `mark_registered` rather than setting `db_id = None` after the fact
- Verify `mark_fired()` is called from `dispatch()` and nowhere else
